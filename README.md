# Sistema de Inventario - Tupac Supermercado Mayorista

Sistema de gestión de inventario para supermercado mayorista en Argentina.

## Demo

Esta es una versión de demostración con datos mock para presentación al cliente.

## Características Implementadas

- Login funcional (usuario: `admin`, contraseña: `demo123`)
- Dashboard con métricas y gráficos
- 40 productos argentinos pre-cargados
- Sistema de alertas
- Control de lotes y vencimientos
- Movimientos de entrada/salida

## Tecnologías

- React 18 + TypeScript
- Vite
- TailwindCSS
- shadcn/ui
- React Router
- Zustand (estado global)
- Recharts (gráficos)
- Lucide React (iconos)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en tu navegador.

## Credenciales de Acceso

- **Usuario**: admin

## Estructura del Proyecto

```
src/
├── components/      # Componentes reutilizables
├── pages/          # Páginas de la aplicación
├── data/           # Datos mock
├── stores/         # Estado global (Zustand)
├── types/          # TypeScript types
├── utils/          # Funciones auxiliares
└── App.tsx         # Configuración de rutas
```
